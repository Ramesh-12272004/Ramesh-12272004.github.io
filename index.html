Soil Testing Virtual Laboratory

Live Web-Based Soil Testing Laboratory using IS Code Standards

Developed by Ramesh Tentu (Civil Engineering)

Project Overview

This web application simulates various soil mechanics tests based on IS standards. It helps students perform calculations and analyze soil properties digitally.

Key Features

  • Sieve Analysis
  • Liquid Limit Test
  • CBR Test
  • Triaxial Test
  • Graph generation & reports
